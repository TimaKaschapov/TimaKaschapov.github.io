6.Первичные навыки конструирования

6.1 Методическое указание по созданию анимации вращения

Цель работы:

  • Научиться создавать анимацию для 3D сборки
  • Получить практические навыки создания анимации 3D сборки в программе «КОМПАС»
  • Ознакомиться с возможностями программы «КОМПАС» в части 3D моделирования

Организационные и методические указания: работа проводится после знакомства с интерфейсом программы «КОМПАС» и создания всех требуемых деталей.

Исходные материалы

  • компьютер
  • готовая сборка
  • программа «КОМПАС»

Ход разработки: Для создания анимации требуется сборка, которая разработана таким образом, чтобы детали могли двигаться и взаимодействовать друг с другом.


Для создания анимации нужно загрузить в «КОМПАС» требуемые параметры.Загрузить их можно через главную панель программы: Приложения – Добавить приложения - Animation3D – Animat.rtw




Выйдет сообщение об успешном подключении библиотек:


Перед тем как начать работу с анимацией нужно отключить все сопряжения в сборке


Для перехода в окно для работы с анимацией на главной панели заходим в Приложение – Механика – Механика: Анимация - Механика: Анимация



В окне работы с анимацией предоставлены следующие функции:

  • Анимация – загрузить, сохранить, настроить анимацию.
  • Анимация – загрузить, сохранить, настроить анимацию.
  • Шаги – анимации производятся пошагово, данное окно позволяет работать с шагами анимации.
  • Компоненты и переменные – добавить исключить какую-либо деталь\компонент, входящий в анимацию.
  • Параметры – перемещение вращение или просто свойства анимации.
  • Соударения – указания для деталей, которые должны ударяться.
  • Траектория точки – указать или удалить вершину точки.
  • Воспроизведение – запуск анимации.
  • Справка – руководство по 3д анимации.

Для создания анимации 1 движения хватит 1 шага, поэтому делаем его начальным состоянием нажимаем по шагу ПКМ и выбираем запомнить начальное состояние


Затем выбираем что должно анимироваться, для этого нажмем Компоненты и переменные – Выбрать компоненты – В дереве сборки


Выделяем детали ЛКМ, для удобства разделим сборку через команду сечение модели


Все выделенные объекты будут двигаться соответственно, поэтому выделяем вал и задаем ему вращение Параметры – Вращение – Выбрать ось вращения – в дереве сборки




Все готово. Теперь можно посмотреть анимацию через Воспроизведение – На текущем шаге



Вывод: с помощью 3D Компаса можно создать анимацию моделей сборок.

Анимация разборки происходит пошагово, поэтому для каждого шага нужно выбирать отдельные детали, которые будут анимироваться на данным шаге, для этого нажмем Компоненты и переменные – Выбрать компоненты – В дереве сборки


Выделяем детали ЛКМ


Дальше создадим все последующие шаги разборки








Перед началом создания анимации нужно настроить скорость движения, для это переходим в Анимация – Настройки и задаем расстояние перемещения



Теперь строим траекторию движения деталей через Параметры – Перемещение – Построить траекторию




Нажимаем кнопку Считать положение, чтобы поставить начальную точку


Отводим деталь на нужное расстояние и повторно нажимаем Считать положение и Завершить

Повторяем процедуру с каждой деталью, до тех пор, пока не разберем ее.Также необходимо проверить правильность постройки траектории, выбрав компонент с построенной траекторией


Если линия траектории не соответствует требуемой как показано на рисунке выше (зеленая линия), то ее нужно перестроить, удалив построенную в окне работы с анимацией и в дереве сборки.

В конечном итоге у нас должна получиться разобранный насос


Теперь, что – бы посмотреть результат нужно все собрать, для это переходим в Анимация – Возврат в исходное состояние


Анимация готова. Можно посмотреть через Воспроизведение – Полное



Вывод: с помощью 3D Компаса можно создать анимацию разбора моделей сборок.

6.3 Методическое указание по созданию чертежа

Цель работы:

  • Научиться создавать чертеж для 3D сборки
  • Получить практические навыки создания чертежа 3D сборки в программе «КОМПАС»
  • Ознакомиться с возможностями программы «КОМПАС» в части 2D моделирования

Организационные и методические указания: работа проводится после знакомства с интерфейсом программы «КОМПАС» и создания всех требуемых деталей.

Исходные материалы

  • компьютер
  • готовая сборка
  • программа «КОМПАС»

Ход разработки: Для создания чертежа нужно задать ему вид, который будет на чертеже. Выбираем Ориентация – Настройка


Настраиваем вид, который будет на чертеже, выбираем вид и нажимаем добавить


Вид готов, теперь нужно загрузить настройки для чертежа, для этого создадим чертеж и загрузим настройки





После – выходим из чертежа и заходим снова, чтобы настройки применились.

Теперь загрузим нашу сборку через функцию Стандартные виды с модели на главной панели



В открывшимся окне редактора поставим Вид1 который создали


Данный вид не влезет на чертеж поэтому поменяем формат и ориентацию листа через Дерево чертежа – Листы, меняем А4 на А3 и ориентацию на горизонтальную


Вернемся к виду, если он пропал – повторяем действия стандартные виды с модели, после чего устанавливаем вид на чертеж


Теперь нужно указать все размеры, делать это нужно через главную панель Размеры


Нам нужен линейный размер и диаметральный размер. Линейный размер задается 2 точками, для диаметрального нужно указать измеряемую окружность



Теперь нужно указать все необходимые размеры


Теперь поставим разрезы, для этого установим линию разреза/сечения в меню Обозначения


Сначала устанавливаем 2 точки начала и конца, затем указываем направление как будет отображаться деталь и отключим проекционную связь


Выкл проекционная связь


Изменить масштаб при необходимости


Разместить на чертеже


Указать размеры во внутренней части


Итоговый чертеж
